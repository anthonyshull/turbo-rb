# README

An example [RoR](https://rubyonrails.org/), [Turbo](https://turbo.hotwired.dev/) app.

(Re)set the database:

```
%> bundle exec rails db:reset
```

Start the server:

```
%> bundle exec rails s
```

You will be directed to a project created in the seed file.

You can edit the project (or cancel.)

You can add a phase to the project (or cancel.)

This is done without reloading the page.
The red boxes indicate portions of the page that are dynamically changed via Turbo.
